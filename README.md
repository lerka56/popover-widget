# Popover Widget

[![Build Status](https://github.com/lerka56/popover-widget/workflows/Deploy%20to%20GitHub%20Pages/badge.svg)](https://github.com/lerka56/popover-widget/actions)
[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Live-brightgreen)](https://lerka56.github.io/popover-widget/)

Pure JavaScript implementation of Bootstrap-like popover widget.

## Features

✅ Без зависимостей 

✅ Показ/скрытие по клику

✅ Закрытие при клике вне области

✅ Центрирование по горизонтали над элементом

✅ Настраиваемые заголовок и содержимое

✅ Адаптивное позиционирование

✅ Полное покрытие тестами Jest
